Class Cache​Split​Render​Object 

open class CacheSplitRenderObject: SubTreeRenderObject

Can be used as a wrapper for e.g. calculation heavy CustomRenderObjects which should get their own cache to avoid triggering heavy calculations if other RenderObjects would need an update in a common cache.
